- Home sections -> `H-01..H-08` -> `kmb.ac.id` homepage + admissions + scholarship refs
- Admissions sections -> `A-01..A-10` -> `kmb.ac.id/admissions-aid`
- Scholarships sections -> `S-01..S-08` -> `scholarship.kmb.ac.id`
- Academics sections -> `AC-01..AC-09` -> `kmb.ac.id` + 7 program subdomains + academic calendar
- Research sections -> `R-01..R-06` -> 4 research center pages + related unit links
- About sections -> `AB-01..AB-07` -> history + vision/mission + chairman page
- Leadership sections -> `L-01..L-04` -> chairman page + missing leadership roster evidence
- Student Life sections -> `SL-01..SL-06` -> nav systems links + tutorial pages
- International sections -> `I-01..I-06` -> `international.kmb.ac.id`
- News sections -> `N-01..N-06` -> international news streams + partial media evidence
- Contact sections -> `C-01..C-05` -> homepage/international contact + admissions payment verification channels

Unsupported content guard:
- KPI strips without validated sources: flagged as unsupported.
- Missing policy blocks: marked `[M]` with owner/unblock.
